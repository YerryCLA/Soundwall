# SoundWall

A futuristic soundboard web app. Click buttons to play sounds.

## Live Demo

https://yerrycla.github.io/Soundwall

## Adding Sounds

1. Add MP3 files to `public/sounds/`
2. Update `public/sounds/sounds.json` with the filename
3. Rebuild and deploy

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```
